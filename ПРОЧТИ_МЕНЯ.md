# Джйотиш Наади — генератор гороскопов

Автономное офлайн-приложение для составления персональных астрологических
разборов в традиции Тамильской Наади. Сидерический зодиак (Нирайна), аянамша
Лахири, северо-индийский стиль карты. Оформление повторяет вашу обложку
(«кашмирский» фон, глубокий красный, золото, шрифт Jaipur, текст Arial).

## Как запустить

**Просто двойным кликом** по `index.html` — работает офлайн в браузере.

Либо через локальный сервер (как другие ваши инструменты):
запустите конфигурацию **jyotish-naadi** — откроется `http://localhost:7898`.

## Что делает

1. **Форма** — имя астролога, тип разбора, дата составления, данные рождения
   клиента, ваши заметки и комментарий по упайям.
2. **Город** — начните вводить название: координаты и часовой пояс подставятся
   автоматически. Любое поле (широта/долгота/пояс) можно поправить вручную
   (например, учесть летнее время — программа берёт стандартный пояс).
3. **«Рассчитать и сформировать отчёт»** — считает сидерическую карту:
   Лагна, 9 планет (знак, дом, точный градус, накшатра, пада), статус
   (🟢 сильна / 🔴 ослаблена / ⚪ нейтрально), северо-индийский чарт,
   толкование по разделам, упайи. Ваши заметки подаются отдельным блоком и
   аккуратно структурируются, не искажая смысла.
4. **«Скачать PDF»** — открывает окно печати. Выберите «Сохранить как PDF»
   и формат **A4**. Отчёт свёрстан под печать: обложка, повторяющиеся красные
   полосы и подпись внизу каждой страницы, разделы не разрываются.
5. **Копировать текст**, **Экспорт/Импорт JSON** (проект-файл),
   **Сохранить черновик**, автосохранение каждые 30 с, **отмена/повтор**
   (Ctrl+Z / Ctrl+Y).

## ИИ-улучшение заметок (необязательно)

Кнопка «✨ Улучшить через ИИ» работает **онлайн** при введённом ключе
Anthropic API: исправляет орфографию/грамматику и слог, сохраняя ваши
предсказания. Без сети/ключа текст всё равно структурируется офлайн при
генерации отчёта.

## Точность расчётов

Аналитические формулы высокой точности (Meeus + кеплеровы элементы JPL).
Положения планет — в пределах угловых минут; этого с запасом хватает для
знака, накшатры и пады. Узлы — средние (Раху/Кету). Проверено на эталонных
картах.

## Структура файлов

```
index.html            интерфейс и контейнер отчёта
css/styles.css         оформление (тема обложки + печать A4)
js/ephemeris.js        астрономический движок (сидерические долготы, лагна)
js/astro-data.js       знаки, накшатры, достоинства, база городов
js/chart.js            построение карты + северо-индийский SVG
js/interpret.js        генерация толкования и упай на русском
js/app.js              форма, расчёт, PDF, JSON, автосохранение, отмена/повтор
fonts/                 Jaipur (заголовки)
```

Код модульный — справочные таблицы и правила оценки статуса
(`STATUS_RULES`) вынесены отдельно, чтобы их легко расширять.
